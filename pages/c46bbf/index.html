使用循环求余判断4的幂

# 题目

给定一个整数，写一个函数来判断它是否是 4 的幂次方。如果是，返回 true ；否则，返回 false 。

整数 n 是 4 的幂次方需满足：存在整数 x 使得 n == 4^x

# 示例

输入：n = 16
输出：true

输入：n = 5
输出：false

输入：n = 1
输出：true

# 解题

4的幂: 可以被x个4整除的数，使用while循环就行

/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfFour = function(n) {
    if (n >= 1) {
        while(!(n % 4)) {
            n = Math.floor(n / 4)
        }
        return n === 1;
    }
    return false;
};
