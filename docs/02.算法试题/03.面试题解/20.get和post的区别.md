## 临时的答案
+ 用法上：`GET`一般用于获取数据；`POST`一般用于新增数据，`POST`也可以用于获取数据
+ 参数形式上：`GET`的参数使用`?`接在`path`后面，多参数之间使用`&`连接；`POST`的参数添加到`request`中以对象的形式传递；
+ 参数传递的字符大小不同：`GET`比`POST`小
+ 安全性上：`GET`参数直接写在`url`上，安全性较低
