## 选择器权重优先级
1. !important， 大过了其它任何设置
2. 内联样式，如: style=””
3. ID选择器，如：#content
4. 类，伪类和属性选择器，如.content
5. 类型选择器、伪元素选择器，如div p
6. 通配符、子选择器、相邻选择器等的。如*、>、+
7. 继承的样式没有权值

> 同一种选择器，级联越多的，权重优先，如 p.p.p > p.p
